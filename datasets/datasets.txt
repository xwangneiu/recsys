Function
utility_matrix(data_csv, output_csv)

Effects
Imports u.data file from MovieLens 100k dataset:
Converts it into a comma-delimited CSV, movielens_utility_matrix.csv.
Rows represent a user's ratings of all movies; columns represent all ratings of a movie

Parameters
data_csv - A string, the path/filename of the source u.data file
output_csv - A string, the path/filename of the destination csv file for the utility matrix

Returns
void
